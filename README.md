##Commands

### 1 Show Databases
  		show dbs	
### 2 Create Database
  		use  database-name 
### 3 Drop Database
     db.dropDatabase()

  		show collections
### 2 Create Collection
  		db.createCollection(collection-name, { capped : true, autoIndexId : true, size : 6142800, max : 10000 } )
### 3 Drop Collection
     db.COLLECTION_NAME.drop()
